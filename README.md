# Cờ Vua Thông Thái

Website giới thiệu chương trình dạy cờ vua cho trẻ em từ 3-6 tuổi.

## Cấu trúc dự án

```
├── index.html          # Trang chủ
├── contact.html        # Trang liên hệ
├── css/
│   ├── style.css      # CSS chính
│   └── contact.css    # CSS trang liên hệ
├── js/
│   └── script.js      # JavaScript
└── README.md          # File hướng dẫn
```

## Tính năng

- Trang chủ với các phần:
  - Banner giới thiệu
  - Mục tiêu chương trình
  - Phương pháp SKGR
  - Lợi ích cho trẻ
  - Sứ mệnh
  
- Trang liên hệ với:
  - Form đăng ký
  - Thông tin liên hệ
  - Hotline: **0968998610**

## Hướng dẫn sử dụng

1. Mở file `index.html` trong trình duyệt để xem trang chủ
2. Điều hướng qua các trang khác nhau thông qua menu
3. Sử dụng form liên hệ để đăng ký thông tin
